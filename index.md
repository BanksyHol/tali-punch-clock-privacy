Privacy Policy – Tali Punch Clock Extension

Last updated: 2026-01-03

Overview

The Tali Punch Clock Chrome extension is designed to allow users to punch in and punch out and record timekeeping events to a Google Sheet. The extension operates only when the user interacts with it and only on supported clinical web applications.

Data Collected

The extension collects only the minimum data required for its functionality:

Doctor name or identifier

Punch in and punch out timestamps

Selected clinic or account (if applicable)

No browsing history, unrelated website data, or personal data beyond timekeeping information is collected.

Data Usage

Collected data is used solely for:

Recording punch in and punch out events

Displaying current punch status within the extension

Sending timekeeping data to a user-configured Google Sheets endpoint

The data is not used for advertising, analytics, profiling, or tracking.

Data Storage

Local configuration and state (such as UI preferences and active punch status) are stored using Chrome’s local storage.

Punch events are transmitted directly to a Google Sheets endpoint controlled by the extension owner.

Data Sharing

The extension:

Does not sell user data

Does not share user data with third parties

Does not transmit data to advertisers or analytics platforms

Data is sent only to the configured Google Sheets destination for timekeeping purposes.

Permissions

The extension uses the following permissions:

storage – to save user configuration and punch state

activeTab – to operate only on the current tab after user interaction

scripting – to inject the punch clock interface

alarms – to support automatic punch-out safeguards

No clipboard access or remote code execution is used.

Security

All network communication is limited to sending and retrieving timekeeping data. The extension does not execute remote code and does not monitor user activity outside its intended scope.

Changes

This privacy policy may be updated if the extension’s functionality changes. Updates will be reflected on this page.

Contact

For questions or concerns about this privacy policy, please contact the extension owner.
